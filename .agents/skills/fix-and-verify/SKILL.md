# Skill: Fix and Verify

## Name
fix-and-verify

## Description
Applies one scoped fix and verifies it with the required checks.

## Goal
Make the smallest possible correction, then validate it with targeted tests or checks.

## Procedure
1. Read the task or issue description.
2. Identify the minimal files in scope.
3. Apply the fix.
4. Run the required validation.
5. Report proof and residual risk.

## Constraints
- **Stay within task scope**.
- **No unrelated refactors**.
- **Do not suppress failing tests**.

## Output Format
- **Fix Applied**
- **Validation**
- **Proof**
- **Remaining Risk**
