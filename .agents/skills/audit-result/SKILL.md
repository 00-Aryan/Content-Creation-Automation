# Skill: Audit Result

## Name
audit-result

## Description
Transforms security or architecture audit notes into a structured result.

## Goal
Take findings from a completed audit and express them as clear, prioritized, actionable issues.

## Procedure
1. Read the source audit carefully.
2. Extract each confirmed finding.
3. Classify by severity.
4. Add evidence, impact, and recommendation.
5. Produce a backlog-ready summary.

## Constraints
- **No code changes**.
- **No remediation implementation**.
- **Keep output evidence-based**.

## Output Format
- **Finding**
- **Severity**
- **Evidence**
- **Impact**
- **Recommendation**
